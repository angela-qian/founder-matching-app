# Founder Match

A Tinder-style app to connect student founders with cofounders.

## Tech Stack
- Next.js + TypeScript
- Tailwind CSS
- PostgreSQL + Prisma
- [Add authentication service later]

## Getting Started
1. Clone the repo
2. `npm install`
3. Copy `.env.example` to `.env` and add your database URL
4. `npx prisma db push`
5. `npm run dev`
